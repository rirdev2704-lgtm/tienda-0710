# ShopMaster - Tienda Virtual

## Descripción del Proyecto
ShopMaster es una tienda virtual moderna y responsiva desarrollada como un proyecto educativo utilizando HTML, CSS, JavaScript, Bootstrap y Font Awesome. Este proyecto consume la API pública [https://fakestoreapi.com/products](https://fakestoreapi.com/products) para mostrar productos en tarjetas con imágenes, títulos, descripciones y precios. Incluye un carrito de compras, un modal para detalles de productos, una pasarela de pago simulada y la generación de un ticket en PDF estilo térmico. El objetivo es simular una compra completa y demostrar el uso de tecnologías web básicas con apoyo de IA.

## Características
- **Tarjetas de Productos**: Muestra productos con imágenes, títulos y precios desde la API.
- **Modal de Detalles**: Permite ver detalles ampliados y agregar productos al carrito.
- **Carrito de Compras**: Gestiona productos añadidos, permite modificar cantidades y eliminar ítems.
- **Pasarela de Pago Simulada**: Incluye un formulario básico con validación.
- **Ticket en PDF**: Genera un recibo imprimible estilo térmico usando jsPDF.
- **Diseño Responsivo**: Adaptado para móviles, tablets y desktops con Bootstrap.

## Instalación
1. Clona el repositorio o descarga los archivos en tu computadora:
   - Crea una carpeta `ShopMaster` en tu escritorio.
   - Copia los archivos `index.html`, `styles.css` y `script.js` en esa carpeta.
2. Instala un editor de código como [Visual Studio Code](https://code.visualstudio.com/).
3. Instala la extensión **Live Server** en VS Code:
   - Busca "Live Server" en las extensiones y haz clic en "Instalar".
4. Abre el proyecto:
   - Abre `index.html` con Live Server (haz clic derecho y selecciona "Open with Live Server").
5. Verifica que funcione en tu navegador (ej., Chrome o Firefox).

## Uso
1. Navega por los productos y haz clic en "Ver Más" para ver detalles.
2. Agrega productos al carrito desde el modal.
3. Revisa el carrito, ajusta cantidades o elimina ítems.
4. Completa el formulario de pago para simular una compra.
5. Descarga el ticket en PDF tras confirmar el pago.
